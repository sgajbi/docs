# Operations

This folder contains the operating standards for keeping the documentation repository clean, source-safe, navigable and reusable.

| File | Purpose |
|---|---|
| [`documentation-governance.md`](documentation-governance.md) | Intake workflow, classification rules, quality rules and product-pack enhancement checklist. |

## Operating Principle

The repository should consolidate useful knowledge into durable curated guides. It should not accumulate raw source packs, local paths, generated-source headers, duplicate material or narrow preparation-only framing.
