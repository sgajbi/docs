## Review Checklist

Before adding a new row, confirm:

1. The source has been classified as product, technical, prompt, reference or operations content.
2. Existing curated material has been checked for overlap.
3. Duplicate or weaker source material has not been copied into the repo.
4. Useful ideas have been rewritten into neutral practitioner guidance.
5. Navigation and coverage files were updated when curated content changed.
6. Local paths, package names, generated headers and preparation-only framing were removed.
