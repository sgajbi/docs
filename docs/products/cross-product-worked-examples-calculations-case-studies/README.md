## Core Principle

A strong wealth platform should be tested through realistic financial examples, not only technical unit tests.

Good examples should show:

```text
business event
-> transaction records
-> position impact
-> cash impact
-> valuation impact
-> performance impact
-> risk impact
-> reporting output
-> controls and QA assertions
```
