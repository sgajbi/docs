# 04 — Real Assets: Real Estate, Infrastructure and Natural Resources

## 1. What are real assets?

Real assets are investments linked to physical or long-lived economic assets such as real estate, infrastructure, energy, timberland, farmland, transport assets and natural resources.

They may be held through:

- private closed-end funds;
- open-ended or evergreen funds;
- REITs/listed vehicles;
- infrastructure funds;
- direct property or project SPVs;
- private credit secured by real assets;
- structured products referencing real-asset indices or funds.

For wealth platforms, real assets combine fund mechanics with property/project-level valuation, income distributions, leverage and long-term cashflow modelling.

## 2. Real estate strategies

| Strategy | Description | Key risks |
|---|---|---|
| Core | Stabilised properties, high occupancy | Interest-rate and valuation risk |
| Core plus | Stabilised plus light improvement/leverage | Leasing, financing, moderate value-add risk |
| Value-add | Improve/lease/reposition properties | Execution, capex, leasing risk |
| Opportunistic | Development, distressed, heavy repositioning | High execution and market-cycle risk |
| Development | Build new properties | Construction, permitting, cost overrun |
| Real estate debt | Loans secured by property | Borrower/refinancing/collateral risk |
| REIT/listed real estate | Listed shares/units | Public market volatility |

## 3. Property types

| Property type | Key drivers |
|---|---|
| Office | Occupancy, lease terms, remote work trends, cap rates |
| Retail | Tenant sales, footfall, lease turnover, e-commerce impact |
| Residential/multifamily | Rental demand, affordability, regulation |
| Industrial/logistics | E-commerce, supply chain, location |
| Data centres | Power, cooling, cloud demand, technology risk |
| Hospitality | Occupancy, room rates, tourism/cycle |
| Healthcare/life sciences | Demographics, specialised demand |
| Student housing | University demand, occupancy |

## 4. Real estate metrics

| Metric | Meaning |
|---|---|
| NOI | Net operating income |
| Cap rate | NOI / property value |
| Occupancy | Leased area / rentable area |
| WALE/WALT | Weighted average lease expiry/term |
| LTV | Debt / property value |
| DSCR | Cashflow / debt service |
| Development yield | Stabilised NOI / development cost |
| IRR | Money-weighted return over project life |
| Equity multiple | Total distributions + NAV / equity invested |
| Appraisal value | Valuation by appraiser/manager |

## 5. Infrastructure strategies

Infrastructure investments finance or own essential assets and services.

| Infrastructure type | Examples | Return drivers |
|---|---|---|
| Core infrastructure | Regulated utilities, toll roads, airports | Stable cashflows, regulation, inflation linkage |
| Core plus | Operational assets with some growth/capex | Cash yield + growth |
| Value-add infra | Assets needing expansion/repositioning | Operational improvement |
| Opportunistic infra | Development/greenfield projects | Construction and demand risk |
| Digital infrastructure | Data centres, fibre, towers | Demand growth, capex, power |
| Energy transition | Renewables, storage, grid | Policy, power pricing, technology |
| Transport | Ports, airports, toll roads | Volume, GDP, regulation |
| Social infrastructure | Hospitals, schools, public assets | Contractual availability payments |

## 6. Infrastructure risks

| Risk | Explanation |
|---|---|
| Regulatory risk | Tariffs, allowed returns or concessions may change |
| Political risk | Government action affects asset value |
| Demand risk | Traffic/volume lower than expected |
| Construction risk | Cost overruns/delays for greenfield projects |
| Counterparty risk | Offtaker/concession counterparty weakness |
| Interest-rate risk | Long-duration cashflows sensitive to rates |
| FX risk | Local revenues vs investor currency |
| ESG/environmental risk | Permits, emissions, climate impact |
| Technology risk | Obsolescence in digital/energy assets |

## 7. Natural resources

Natural resources may include energy, mining, timberland, farmland and commodities-linked assets.

| Segment | Drivers |
|---|---|
| Energy | Commodity prices, reserves, capex, regulation |
| Mining | Commodity cycle, grades, extraction cost |
| Timberland | Biological growth, timber prices, land value |
| Farmland | Crop prices, yield, weather, water rights |
| Royalties | Production volumes and prices |

These assets can provide inflation linkage but introduce commodity, environmental, operational and regulatory risks.

## 8. Real-asset lifecycle

Real assets often follow a long-cycle lifecycle:

```text
Fundraise -> acquire/develop asset -> stabilise/operate -> distribute income -> refinance/optimise -> exit/sell -> final distributions
```

For development or value-add:

```text
Land/acquisition -> permits -> construction/capex -> leasing -> stabilisation -> exit/refinance
```

## 9. Real-asset transaction modelling

At fund-investor level:

| Transaction type | Use |
|---|---|
| ALT_COMMITMENT | Commitment to fund/SPV |
| ALT_CAPITAL_CALL | Capital call notice |
| ALT_CONTRIBUTION | Cash paid |
| ALT_DISTRIBUTION_INCOME | Rental/operating income distribution |
| ALT_DISTRIBUTION_ROC | Return of capital |
| ALT_DISTRIBUTION_GAIN | Realised gain on asset sale |
| ALT_NAV_UPDATE | Periodic valuation update |
| ALT_SECONDARY_SELL | Sale of fund interest |
| ALT_FINAL_LIQUIDATION | Final close-out |

At direct property/SPV level, also model:

- property acquisition;
- capex contribution;
- debt drawdown/repayment;
- rental income;
- operating expenses;
- valuation/appraisal update;
- property sale;
- tax and withholding.

## 10. Valuation of real assets

Real assets often use appraisal and DCF methods rather than exchange prices.

| Method | Use |
|---|---|
| Income capitalisation | Stabilised real estate using NOI and cap rate |
| Discounted cashflow | Project future cashflows and exit value |
| Comparable transactions | Similar asset sales |
| Replacement cost | Useful for certain property/infrastructure assets |
| NAV from manager | Fund-level reporting |
| Independent appraisal | Periodic appraisal support |

Key valuation variables:

- occupancy;
- rent growth;
- cap rates;
- discount rate;
- leverage;
- refinancing assumptions;
- exit value;
- inflation;
- regulatory/tariff regime;
- construction cost;
- operating expenses;
- FX.

## 11. Advisory considerations

Real assets are often positioned as income, inflation-sensitive or diversification assets, but clients must understand:

- income may not be stable;
- capital values are interest-rate sensitive;
- valuations may be lagged;
- leverage may magnify losses;
- liquidity is limited;
- property cycles can be long;
- development and opportunistic strategies can be high risk;
- listed REIT exposure behaves more like public equity in market stress;
- private real estate funds may suspend or gate redemptions.

## 12. Reporting recommendations

Real-asset reporting should include:

| Report item | Why it matters |
|---|---|
| NAV date and lag | Valuation freshness |
| Fund strategy | Core/core-plus/value-add/opportunistic |
| Property type exposure | Office/retail/residential/logistics/etc. |
| Geography | Country/city concentration |
| Leverage | Fund/property debt risk |
| Occupancy/WALE | Income durability |
| Income versus capital return | Distribution quality |
| Unfunded commitment | Future cash obligation |
| Vintage/year of investment | Cycle exposure |
| FX exposure | Currency impact |
| Liquidity terms | Redemption/lock-up/gates |
