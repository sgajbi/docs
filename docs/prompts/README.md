# Prompt Documentation

Prompt documentation belongs here.

Use this section for:

1. reusable prompt packs,
2. product-specific prompt instructions,
3. evaluation rubrics,
4. agent operating patterns,
5. examples of good and bad outputs,
6. model or workflow notes that should be preserved.

## Recommended Structure

```text
docs/prompts/<prompt-pack-slug>/
  README.md
  placeholder-guide.md
  quick-use-menu.md
  copy-paste-workflows.md
  prompts.md
  evaluation.md
  examples.md
```

Keep prompts separate from product reference material unless a document is explicitly a product-specific prompt pack.
