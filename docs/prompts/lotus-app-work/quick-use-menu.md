## Common Combinations

For a new RFC:

1. Phase 0 Ramp-Up
2. Gold-Standard RFC Planning
3. RFC Implementation Loop after approval
4. PR Loop And Branch Hygiene
5. Post-Completion Retrospective

For backend hardening:

1. Phase 0 Ramp-Up
2. Backend Bank-Buyable Refactor for focused slices, or Enterprise Backend Refactoring Program for a large multi-commit refactor
3. Endpoint Certification if APIs changed
4. Documentation, Context, Wiki, And Skill Sync
5. PR Loop And Branch Hygiene

For UI work:

1. Phase 0 Ramp-Up
2. Workbench UI Refactor And Audit
3. Demo And Live Runtime Proof
4. PR Loop And Branch Hygiene

For cross-app realization:

1. Phase 0 Ramp-Up for every touched app
2. RFC Implementation Loop with app-specific ownership
3. Endpoint Certification for changed API families
4. Demo And Live Runtime Proof if Workbench changed
5. PR Loop And Branch Hygiene for each repo
