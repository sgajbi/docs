# Reference Material

Reference material belongs here.

Use this section for:

1. external source lists,
2. citations and links,
3. glossary files,
4. regulatory or investor-education references,
5. reusable background material that supports multiple document packs.

When a reference only supports one product pack, it can remain inside that product pack. Promote it here when it becomes reusable across multiple topics.
