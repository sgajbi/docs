# 04 — Bonds Addendum: Advisory, Mandate, Analytics and Reporting

## 1. Advisory questions for bonds

| Area | Question |
|---|---|
| Objective | Income, capital preservation, duration exposure, credit spread exposure? |
| Credit quality | Is issuer rating/credit risk suitable? |
| Duration | Can client tolerate interest-rate sensitivity? |
| Currency | Does bond currency match client needs? |
| Liquidity | Can the bond be sold before maturity? |
| Callability | Can issuer call the bond and reduce expected yield? |
| Yield source | Is yield from rates, credit spread, discount, complexity or illiquidity? |
| Seniority | Senior, subordinated, AT1, perpetual? |
| Concentration | Issuer, sector, country, rating bucket |
| Tax | Coupon withholding/tax treatment |

## 2. Mandate controls for bonds

| Control | Example |
|---|---|
| Minimum rating | A-/BBB-/investment grade only |
| HY/unrated limit | Max 10% high yield, max 5% unrated |
| Issuer limit | Max 5% per issuer |
| Sector/country limit | Financials/EM exposure controls |
| Duration range | 2-6 years |
| Maturity ladder | Avoid maturity cliff |
| Currency limit | Non-base currency max 30% |
| Subordinated/AT1 limit | Max 5% or prohibited |
| Callable/perpetual control | Suitability and yield-to-call disclosure |

## 3. Analytics for bonds

| Analytics | Description |
|---|---|
| Yield to maturity | Return if held to maturity and no default |
| Yield to call/worst | Important for callable bonds |
| Duration | Interest-rate sensitivity |
| DV01 | Price impact of 1bp yield move |
| Spread | Credit risk compensation |
| Accrued interest | Earned interest since last coupon |
| Clean/dirty price | Quoted vs settlement value |
| Rating distribution | Credit quality |
| Maturity ladder | Reinvestment/liquidity planning |
| Income forecast | Expected coupons |

## 4. Reporting sections for bonds

| Section | Content |
|---|---|
| Holdings | Issuer, rating, maturity, coupon, clean/dirty price, market value |
| Income | Coupon schedule, accrued, income forecast |
| Risk | Duration, rating, spread, currency, liquidity |
| Maturity ladder | Upcoming maturities and reinvestment need |
| Events | Calls, puts, maturities, rating changes |
| Attribution | Carry, yield curve, spread, FX, selection |

## 5. Reporting warning

Do not show only high yield. Explain whether yield comes from credit risk, long duration, call risk, subordination, currency risk or illiquidity.
