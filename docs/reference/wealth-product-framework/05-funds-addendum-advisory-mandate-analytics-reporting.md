# 05 — Funds Addendum: Advisory, Mandate, Analytics and Reporting

## 1. Advisory questions for funds

| Area | Question |
|---|---|
| Objective | Growth, income, diversification, alternative exposure? |
| Strategy | What does the fund invest in and how? |
| Manager | Is the manager/fund house approved and credible? |
| Fees | TER, management fee, performance fee, entry/exit fee? |
| Liquidity | Daily dealing, monthly, gated, locked-up? |
| Share class | Accumulating/distributing, hedged/unhedged, retail/institutional? |
| Risk | Asset class, volatility, drawdown, leverage, derivatives? |
| Benchmark | What should performance be compared to? |
| Concentration | Fund house, strategy, underlying exposure |
| Tax/currency | Share class currency and withholding/distribution treatment |

## 2. Mandate controls for funds

| Control | Example |
|---|---|
| Fund eligibility | Approved fund list only |
| Fund house limit | Max exposure to one manager/fund house |
| Strategy limit | Alternatives max 15%, hedge funds max 10% |
| Liquidity bucket | Illiquid/monthly funds max 10% |
| Share class control | Use institutional/clean share classes where eligible |
| Currency hedging | Hedged share class required for some mandates |
| Look-through limits | Underlying equity/FI/country/sector exposure |
| Fee control | Avoid high-fee classes if cheaper eligible class available |

## 3. Analytics for funds

| Analytics | Description |
|---|---|
| NAV return | Unit NAV movement plus distributions |
| Distribution yield | Income distribution relative to NAV |
| TER/ongoing charges | Cost drag |
| Alpha/beta | Manager value-add vs benchmark |
| Tracking error | Active risk |
| Sharpe/Sortino | Risk-adjusted return |
| Drawdown | Downside risk |
| Look-through allocation | Underlying sectors/countries/assets |
| Manager concentration | Exposure by fund manager |
| Liquidity profile | Dealing frequency, gates, lockups |

## 4. Reporting sections for funds

| Section | Content |
|---|---|
| Holdings | Units, NAV, market value, cost, P&L |
| Fund profile | Strategy, manager, benchmark, share class |
| Performance | Fund return vs benchmark/category |
| Income | Distributions, reinvestments, yield |
| Fees | TER and explicit transaction/platform fees |
| Risk | Volatility, drawdown, liquidity, strategy risk |
| Look-through | Asset, sector, country, currency exposure |
| Events | Distribution, fund merger, suspension, gate, strategy change |

## 5. Reporting warning

Do not treat fund distribution as guaranteed income. Some distributions may come from income, capital gains or capital. Reporting should distinguish distribution policy from actual economic return.
