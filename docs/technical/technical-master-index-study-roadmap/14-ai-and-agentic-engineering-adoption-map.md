## AI Feature Review Questions

- What use case is supported?
- What is the risk tier?
- What data is accessed?
- Are sources approved?
- Is retrieval entitlement-aware?
- Is output grounded?
- Is human review needed?
- What tools can the agent call?
- Are tool actions authorized and audited?
- How is output tested?
- How is prompt/model version observed?
- What happens if AI output is wrong?

## Coding Agent Adoption

Use coding agents for:

- refactoring
- test generation
- documentation
- quality gates
- runbooks
- evidence packs
- review preparation

Guardrails:

- repo context file
- agent operating contract
- small commits
- CI validation
- human review
- no secrets
- no unsupported claims
- evidence summary

---

## Summary

AI should amplify engineering standards, not bypass them.

Adopt AI through governed use cases, source control, testing, evidence, and human oversight.
